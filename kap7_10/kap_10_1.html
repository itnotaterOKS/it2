Skjemaer med HTML (og php)

Taggen for skjema er < form > < /form >, denne brukte vi i IT-1 for å kunne hente informasjon fra skjemaer med php. Når vi bearbeider skjemaer med JavaScript kommer vi ikke til å bruke form. Mellom her angir vi de inputene vi ønsker at brukeren skal fylle ut. I < form > er det viktigste attributtet method: dette sier noe om hva vi skal gjøre med inndataen. Vi bruker GET når det ikke er sensitiv data og POST når det er sensitiv data. I IT-2-faget vil vi som oftes ikke ha bruk for form-taggene i det hele tatt, da vi bearbeider skjemaet med js og ikke php.

Koden for et skjema i IT-2 kan derfor være

   < div class="skjema" >
     Skriv inn navn: < input type = "text" name = "navn" > < br >
     Skriv inn alder: < input type = "number" name = "alder" > < br >
     Hva synes du om denne nettsiden? < select name="tilbakemelding" >
     Kan du utdype dette?
     < textarea name="name" rows="4" cols="50" placeholder="skriv inn dine synspunkter om denne siden" >< /textarea >< br>
       < option value="1" > Dårlig < /option >
       < option value="2" > Helt OK < /option >
       < option value="3" > Kjempebra! < /option >
     < /select > < br >
     Kjønn: < input type = "radio" name = "kjonn" > Kvinne < input type="radio" name="kjonn" > Mann < br >
     Hva liker du å spise til middag?
     < input type="checkbox" name="middag" value="pasta"> Pasta < br >
     < input type="checkbox" name="middag" value="pizza"> Pizza < br>
     < input type="checkbox" name="middag" value="taco"> Taco < br >
     Og hva er favoritten din av disse?
     < input type="radio" name="favmiddag" value="pasta"> Pasta < br >
     < input type="radio" name="favmiddag" value="pizza"> Pizza < br >
     < input type="radio" name="favmiddag" value="taco"> Taco < br>
     < input type="submit" name="send" value="Send tilbakemelding" >
   < /div >

Mer om skjemaer kan du lese på w3schools. Læreboka har et greit css-dokument for skjemaer, dette kan vi laste ned her.

Oppgave

Last ned css-dokumentet fra læreboka og flytt det inn i prosjektmappen din for faget. Lag et skjema som inneholder "alle" typer inputfelt og bruk dette til å utforskje stilarket du lastet ned. Gjør de endringene du ønsker sånn at dette kan være ditt skjema-stilark for alle fremtidige nettsider med brukerinput.
