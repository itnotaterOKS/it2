Position

Hva?
Strukturerer posisjonen til et element på en nettside

Hvordan?
Bruker egenskapen position, som kan ha ulike verdier

Minner i tillegg om at alle elementer enten har display: inline (ved siden av, f.eks. < img > og < span >) eller block (under, f.eks < li > og < div >)

verdi beskrivelse
static default verdi, elementet følger rekkefølgen det legges i koden
absolute elementet ligger inne i et annet element, posisjonen justeres i forhold til dette
relative posisjonen sammenliknes med der det normalt ville vært plassert
fixed posisjonen bestemmes relativt til vinduet

Eksempel: å bruke css og js til å la et element følge musepekeren

I html-koden:
  < div id="folgerPil" > Følger musepeker! < /div >

Følger musepeker!

I css-koden:
   #folgerPil{
     border: 1px solid red;
     width: 150 px;
     height: 50 px;
     border-radius: 50%;
     position: fixed;
     left: 0;
     top: 0;
  }

I js-koden
  < script >
     var bodyEl = document.querySelector("body"); //henter element
     var mouseEl = document.querySelector("#folgerMus"); //henter element
     bodyEl.addEventListener ("mousemove", flyttSirkel); //legger til lytte
     function flyttSirkel(e) { // hendelsesstyrt funksjon
       mouseEl.style.left = (e.clientX - 25) + "px"; //x-posisjon
       mouseEl.style.top = (e.clientY - 25) + "px"; //y-posisjon
       var h = (e.clientX/e.clientY)*200 //farge er avhenging av posisjonen
       mouseEl.style.backgroundColor = " hsl(" + h + ",70% , 50%) ";
     }
   < /script >

Eksempel: å bruke css + js til å styre et element med piltastene

Ganske liknende eksempel som forrige, men vil her la elementet følge piltaster

I html-koden
< div id="folgerPil" >Følger piltaster!< /div >

Følger piltaster!

I css-koden
   #folgerPil{
     border: 1px solid red;
     width: 150 px;
     height: 50 px;
     position: fixed;
     left: 0;
     top: 0;
  }

I js-koden
   var bodyEl = document.querySelector("body"); //henter element
   var pilEl = document.querySelector("#folgerPil"); //henter element
   var venstre = 0; //setter avstandsvariabel
   var topp = 0; //setter avstandsvariabel
   var dist = 10; //hvor langt den flyttes

     bodyEl.addEventListener ("keydown", flyttFirkant); //legger til lytter
     function flyttFirkant(e) { //hendelsesstyrt funksjon
     if (e.keyCode === 37) { //pil venstre
       venstre -= dist; //flytter mot venstre
    }
    else if (e.keyCode === 39) { //pil høyre
      venstre += dist; //flytter mot høyre
    }
     else if (e.keyCode === 38) { //pil opp
      topp -= dist; //flytter opp
    }
     else if (e.keyCode === 40) { //pil ned
       topp += dist; //flytter ned
    }
    pilEl.style.top = topp + "px"; //setter ny avstand
    pilEl.style.left = venstre + "px"; //setter ny avstand
  }
